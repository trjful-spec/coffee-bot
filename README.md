# Coffee Bot (@Coffee_kohi_Bot)

Автоматизированный Telegram-бот на базе фреймворка Aiogram 3, предназначенный для координации совместных встреч и организации опросов в супергруппах.

---

## Warning
Описание сгенерировано ИИ

---

## Основная задача бота

Главной функцией бота является автоматизация контроля за ходом голосования. Архитектура приложения включает встроенный фоновый планировщик, который отслеживает участников, выбравших вариант «Отвечу позже». 

Бот автоматически вычисляет динамические дедлайны на основе настроек конкретного чата, осуществляет регулярное упоминание (тегирование) неопределившихся пользователей по их Telegram ID для побуждения к выбору и отправляет финальное предупреждение за 10 минут до окончания сбора ответов.

---

## Функциональные возможности

* **Управление дедлайнами:** Автоматический расчет времени закрытия опроса и промежуточных напоминаний.
* **Автоматическое завершение:** Блокировка инлайн-кнопок и обновление итогового текста опроса по наступлении времени встречи.
* **Очистка закрепленных сообщений:** Автоматическое открепление (unpin) сообщения с опросом из чата через 2 часа после завершения встречи.
* **Разграничение прав:** Изменение параметров опроса (время, интервалы) доступно только инициатору встречи или администраторам группы.

## Командный интерфейс

| Команда | Описание | Пример использования |
| :--- | :--- | :--- |
| `/coffee` | Создать опрос с дефолтными настройками | `/coffee` |
| `/coffee [HH:MM]` | Создать опрос на указанное время (место по умолчанию) | `/coffee 15:30` |
| `/coffee [HH:MM] [Место]` | Создать опрос с указанием времени и точки сбора | `/coffee 15:30 Мега` |
| `/close` | Досрочно закрыть голосование и зафиксировать результаты | `/close` |
| `/cancel` | Отменить и удалить текущее голосование | `/cancel` |
| `/time [HH:MM]` | Изменить время встречи в активном голосовании | `/time 20:00` |
| `/interval [N]` | Изменить интервал напоминаний (в часах до встречи) | `/interval 2` |

## Архитектура и стек технологий

* **Язык программирования:** Python 3.10+
* **Фреймворк:** Aiogram 3.x (Asynchronous Telegram Bots API)
* **Асинхронность:** Asyncio (управление фоновыми задачами через `asyncio.create_task`)
* **База данных:** SQLAlchemy / SQLite (асинхронный режим)
* **Управление конфигурацией:** Ansible

## Развертывание с помощью Ansible

Развертывание и конфигурация бота на целевом сервере полностью автоматизированы с помощью Ansible. 

### Важное требование: Настройка Timezone
Поскольку логика вычисления дедлайнов опирается на системное время, критически важно, чтобы часовой пояс целевого сервера совпадал с часовым поясом пользователей (например, `Europe/Moscow` для UTC+3). В противном случае расчет `time_left` внутри фонового воркера будет производиться некорректно, что приведет к сдвигу или пропуску уведомлений. Настройка таймзоны интегрирована в конфигурацию плейбука.

### Структура Ansible-роли

```text
├── ansible.cfg
├── deploy_from_github.yml
├── deploy_from_local.yml
├── group_vars
│   └── all.yml
├── inventory.ini
├── roles
│   ├── bot
│   │   ├── handlers
│   │   │   └── main.yml
│   │   ├── tasks
│   │   │   └── main.yml
│   │   └── templates
│   │       ├── coffee-bot.service.j2
│   │       └── env.j2
│   └── common
│       └── tasks
│           └── main.yml
├── start_coffee_bot.yml
└── stop_coffee_bot.yml
```

При развертывании из репозитория убедитесь, что заполнены поля файла group_vars/all.yml:
```bash
project_name: coffee-bot
project_path: /opt/coffee-bot
venv_path: /opt/coffee-bot/.venv
python_bin: /usr/bin/python3
bot_user: ubuntu
bot_group: ubuntu
bot_token: "XXX"
default_place: "Мега"
default_time: "21:00"
```
